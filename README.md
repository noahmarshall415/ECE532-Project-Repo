# ECE532-Project-Repo

This repository includes all the necessary source, constraint, block diagram, and memory files used in the project. Also included is an IP repo folder that contains imported IPs, both custom and third-party, that are used as well. The project_1.tcl file can be run in the Vivado terminal to build the project in its entirety. This can be done by navigating to the source directory in the terminal and using the command: “source project_1.tcl”. Additionally, the final group report for the project is located in the doc folder and a few unused or temporary files mentioned in the report are included as well.
